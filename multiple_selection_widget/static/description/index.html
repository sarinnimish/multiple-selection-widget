This module introduces a custom reusable multi-select dropdown widget using the Select2 library. It enhances user experience on website forms by offering a searchable, clearable, multi-select interface.

üîß Key Features

  • Applies to any <select widget="select2"> element in website templates.
  • Supports multiple selection, searching, and clearing of values.
  • Dynamically loads select2.js and select2.css only when needed.
  • Logs selected values to the browser console on change.
  • Reusable in any website QWeb template using template inheritance.

‚öôÔ∏è How It Works

  1. The JavaScript widget (`ReusableSelect2`) extends Odoo's `publicWidget` system.
  2. It automatically detects any <select widget="select2"> element on the page.
  3. If Select2 is not already loaded, it dynamically loads the JS and CSS files.
  4. The dropdown is initialized with placeholder and clearable options.

üìÑ Example Usage

Add this snippet inside any inherited template where you want to use the widget:

<select widget="select2" name="your_field_name" multiple="multiple" data-placeholder="Select options">
  <option value="opt1">Option 1</option>
  <option value="opt2">Option 2</option>
  <option value="opt3">Option 3</option>
</select>

üìÅ File Overview

  • JS: static/src/js/m2m_select2_widget.js
  • Assets: static/lib/select2/ (JS/CSS)
  • Manifest: Includes widget JS in web.assets_frontend

‚úÖ Notes

  • Always add multiple="multiple" to the <select> tag to enable multi-select functionality with Select2.
  • No specific template is required; you can inherit any template where you need the dropdown.
  • Make sure to place the select field within a valid <form> block if submitting data.
  • You can customize the Select2 options further in the JS widget if needed.
